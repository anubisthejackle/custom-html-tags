Custom HTML Item 1 Item 2 Item 3 Welcome to my little test project! Basically, this is a project showing how we might write more semantic HTML. Each tag on this page is a custom HTML tag -- those that aren't required to run it, at least. Styling is done with Tailwind CSS by creating custom components. This allows us to create a minimal HTML front-end, and contain all of the style in our CSS file -- all without using any class names.
